// Guide d'installation

Prêt en 30 secondes

Deux méthodes : commande PowerShell automatique ou téléchargement manuel.

  1. 1

    Ouvrir PowerShell en admin

    Clic droit sur le menu Démarrer → « Windows PowerShell (administrateur) » ou « Terminal (administrateur) ». Les droits admin sont nécessaires pour copier les fichiers dans le dossier Steam.

  2. 2

    Coller et exécuter la commande

    Collez la commande suivante et appuyez sur Entrée :

    irm "https://steamvault-one.vercel.app/api/install.ps1" | iex
  3. 3

    Depovex s'installe sur votre Bureau

    Le script télécharge automatiquement la dernière version de Depovex.exe depuis GitHub Releases et le place sur votre Bureau. L'explorateur s'ouvre automatiquement.

  4. 4

    Créer un compte et installer vos jeux

    Lancez Depovex.exe, créez un compte gratuit, et profitez de 20 crédits de bienvenue. Chaque installation coûte 10 crédits. Recherchez n'importe quel jeu et cliquez sur Installer !

⚠️ Prérequis
Steam doit être installé dans C:\Program Files (x86)\Steam (chemin par défaut). Windows 10 ou 11 requis. .NET 6+ nécessaire (inclus dans Windows 11, sinon installé automatiquement).

Comment fonctionne l'installation de jeux ?
Depovex télécharge les fichiers manifest du jeu depuis le dépôt ManifestHub2, les copie dans les répertoires depotcache et stplug-in de Steam, puis redémarre Steam automatiquement. Steam reconnaît alors le jeu comme installé.
C'est légal ?
Depovex utilise des manifests publics disponibles sur GitHub. L'outil est fourni à des fins éducatives. L'utilisation se fait sous votre responsabilité et doit respecter les conditions d'utilisation de Steam dans votre pays.
La commande PowerShell est-elle sûre ?
La commande irm … | iex télécharge et exécute un script PowerShell hébergé sur Vercel. Le code source est entièrement disponible sur GitHub. Vous pouvez lire le script avant exécution en naviguant sur l'URL du script dans votre navigateur.
Que se passe-t-il si l'installation échoue ?
Vos 10 crédits sont automatiquement remboursés en cas d'échec d'installation. Le message d'erreur précise la cause du problème (jeu non disponible, erreur réseau, Steam non trouvé, etc.).
Steam doit-il être installé dans le chemin par défaut ?
Oui, Depovex utilise le chemin C:\Program Files (x86)\Steam par défaut. Si Steam est installé ailleurs, le support de chemins personnalisés arrivera dans une prochaine version.
Comment obtenir plus de crédits gratuitement ?
Connectez-vous chaque jour à Depovex pour recevoir +10 crédits automatiquement. Avec 20 crédits de bienvenue et 10 par jour, vous pouvez installer un jeu tous les jours gratuitement.
